# BLACKSTONE CRYPTOGRAPHIC LIBRARY v3.0

Military-grade cryptographic library

## Features
- AES-256-GCM
- ChaCha20-Poly1305
- SHA-256 / SHA-512
- HMAC-SHA256
- PBKDF2-HMAC-SHA256
- HKDF
- Constant-time operations
- Secure memory zeroization

## Build
`gcc -O3 -march=native main.c -o blackstone`
