## ✨ FEATURES

| Category | Features |
|----------|----------|
| **Encryption** | AES-256-GCM, ChaCha20-Poly1305 |
| **Hashing** | SHA-256, SHA-512, HMAC-SHA256 |
| **Key Derivation** | PBKDF2-HMAC-SHA256, HKDF |
| **Session Management** | TTL-based sessions, Key rotation |
| **Auto-Updater** | Automatic updates, Rollback support |
| **Integrity** | CRC32, Digital signatures, Checksums |
| **Platform Support** | Windows, Linux, macOS |
